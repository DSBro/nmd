About No More DDOS


No More DDOS is the "10 years later" update of DDOS Deflate. No More DDOS add more functionality to DDOS Deflate with the compatibility for the latest GNU/Linux OS.

 

No More DDOS doesn't need additional dependencies to work, but if you want the mail function to work, you need to have the mail function configured on your server.
